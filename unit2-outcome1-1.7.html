UNIT 2: Outcome 1: 1.7

There are various tool kits that are needed to ensure that a website is able to run on multiple types of browsers such as -webkit- ; -o-; -ms- and -moz- these are the browser engines vendors that allow a web page to open on chrome, Mozilla Firefox, safari and internet explorer. Browser vendors allow new features or modified CSS properties into the specific browser. This is done by prefixing property names with strings that are only known by their respective browser engine. The browser would scan a document and find a CSS property that is prefixed with the string that is known to the respective to the engine that applies it to the current document. This essentially allows the browser vendors to present new features if the specifications are missing.[6]
