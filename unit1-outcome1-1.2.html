To make digital projects successful I would need to be competent in using various programming software and organising software. The use of slack and trello could be very important. Trello can certainly make the management of a project a lot less stressful. it allows the user to visualise a project a lot easier , it allows constructive feedback and it stops the user forgetting tasks. slack is an important platform that allows people to organise , plan and stay in touch with other colleagues.
