UNIT 1: Outcome 2: 2.1

The IT tools that were used during the duration of the project were Visual studio code, Trello, slack, google chrome and Mozilla Firefox. Overall, every one of these tools were very helpful and useful when designing and constructing my site. Trello allowed me to organise and plan the project. It allowed me to keep focussed and whenever I got stuck, I always went back to view my Trello board to put me right back on track to complete the necessary tasks. Slack allowed me to receive course material and help from my course instructor when needed. Visual studio code allowed me to design the html code for all the respective web pages as well as designing the appropriate CSS sheets necessary for designing the pages to my site. I used both Mozilla Firefox and google chrome to view the respective html pages once I had designed them.
