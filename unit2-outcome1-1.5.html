UNIT 2: Outcome 1: 1.5

When creating a site the use of images and videos have to be carefully monitored otherwise the created website would get flagged. To use an image or a video from a source permission to use this data is needed. This permission would need to be gathered from all the copyright owners of the content. This includes “a single image with numerous creators, a licensed image, or an image with embedded copyright works” [4]. If copyright laws have been broken there are many courses of action that would be taken by an “individual or organisation that owns or administers the copyright” [4]. If an image is used users will have to pay for the image license. “However, legal action might be taken by bringing a claim in court which could result in having to go to court for a hearing” [4].
